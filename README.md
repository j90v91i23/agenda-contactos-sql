#Agenda de Contactos – SQL Server

Proyecto de base de datos relacional desarrollado en **SQL Server**, orientado a la gestión de usuarios, contactos, teléfonos, direcciones y recordatorios.

En este proyecto aplico  conocimientos en **modelado entidad–relación**, **consultas**, y **buenas prácticas de bases de datos**.

---

##Descripción del proyecto

La Agenda de Contactos permite:

- Gestión de **usuarios y roles**
- Registro de **contactos asociados a cada usuario**
- Manejo de **múltiples teléfonos por contacto**
- Manejo de **direcciones por contacto**, clasificadas por tipo (casa, oficina, etc.)
- Uso de **eliminación lógica**
- Registro de **recordatorios y tareas**

---
